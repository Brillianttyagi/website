HSN Code and E-commerce

FEB 01, 2021
By - Seventh Square

Harmonised System of Nomenclature (HSN) helps in classifying goods in a systematic and organised manner. It is a six digit uniform code that classifies more than 5,000 products and is accepted worldwide. This uniform code helps in identifying the tax code of a commodity.

How is HSN Code relevant to you when selling with us?

  1. GST Law requires each product to be classified to appropriate HSN Code
  2. Tax code applicable for a given HSN Code will be used for Tax Computing purposes
  3. As a compliance requirement, HSN Code has to be mandatorily printed on all invoices.

How to find applicable HSN Code while adding a product?

HSN Code is based on the Product category and type. It is divided into chapters and subchapters. Your relevant code could be complete 6 digits or just 4 digits. If you don’t already know your product’s HSN Code, it is not an issue. Find your relevant HSN Code using the link next to the field in ‘ Add Product ’ form. We also regularly check products uploaded and will help you in case the code turns out to be wrong.
